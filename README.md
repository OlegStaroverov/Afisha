# MAX Mini App — стартовая версия

Это первая рабочая версия мини-приложения под MAX на чистых HTML, CSS и JS.

## Что уже сделано

- темный мобильный интерфейс в стиле референса
- главная лента событий
- фильтр по дате
- фильтр по площадке
- карточка события
- избранное
- подписки на площадки
- отзывы
- экран уведомлений
- интеграция с `window.WebApp`
- привязка локальных данных к `user.id` из MAX
- deep link поддержка через `startapp=event_1` и подобные параметры

## Файлы

- `index.html` — точка входа
- `styles.css` — стили
- `app.js` — логика приложения

## Как устроена привязка к аккаунту MAX

При запуске приложение читает:

- `window.WebApp.initDataUnsafe.user`
- `window.WebApp.initDataUnsafe.start_param`
- `window.WebApp.platform`
- `window.WebApp.version`

Для демо данные избранного, подписок, отзывов и уведомлений записываются:

1. в `WebApp.DeviceStorage`, если доступно мобильное окружение MAX
2. в `localStorage` как fallback

Ключи сохраняются в формате:

`max-mini-app:<userId>:favorites`

Это значит, что на уровне прототипа состояние уже разделяется по пользователям MAX.

## Что надо сделать следующим этапом

### 1. Подключить backend
Сейчас данные хранятся локально на устройстве. Для продакшна нужно вынести на сервер:

- события
- площадки
- отзывы
- избранное
- подписки
- уведомления

### 2. Валидировать `initData` на сервере
Для безопасной привязки пользователя надо валидировать `window.WebApp.initData` через HMAC-SHA256 по схеме MAX.

### 3. Подключить Bot API
Нужно добавить:

- реальные уведомления от бота
- шаринг контента через `mid`
- переходы по диплинкам на конкретные события

### 4. Подключить реальные карты и API
Сейчас карта в карточке — это визуальный блок. Потом можно сделать:

- переход в карты
- построение маршрута
- геолокацию

## Как запускать локально

Откройте `index.html` через локальный сервер, например:

```bash
python3 -m http.server 8000
```

Потом откройте:

```text
http://localhost:8000
```

## Для публикации в MAX

Нужно загрузить папку на HTTPS-хостинг:

- Vercel
- GitHub Pages
- Netlify

И вставить URL в настройках бота MAX.
